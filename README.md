# 一个功能简单的五子棋
> 作者：Kiprey

> E-mail: Kiprey@qq.com

## 简介
这个游戏是我在上手Qt时做的第一个项目

因为我在学Qt时直接上项目，所以代码可能会有点混乱

AI也是我的第一次尝试，所以肯定会存在代码长但不太智能的现象

该游戏支持AI对战与联机对战，同时还支持悔棋操作。

## 后记
